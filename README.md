# Aussie Tax Legal Database

Exports from the [ATO Legal Database](https://www.ato.gov.au/law/) — automatically scraped and updated whenever new documents are published.

## Download the data

All data files are published as **release assets**. Go to the [Releases page](../../releases/tag/latest-data) to download the latest versions.

| File | Contents |
|------|----------|
| `edited_private_advice_all.zip` | All Edited Private Advice (EPAs) |
| `ato_interpretative_decisions.zip` | ATO Interpretative Decisions (ATOIDs) |
| `public_all.zip` | Public Rulings & Determinations |
| `practical_compliance_guidelines.zip` | Practical Compliance Guidelines (PCGs) |

Each zip contains a single UTF-8 CSV file. Files are updated automatically on weekdays when new ATO documents are detected.
